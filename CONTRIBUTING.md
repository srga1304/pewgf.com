# Contributing to PEWGF Trainer

First off, thank you for considering contributing to PEWGF Trainer! 

## How Can I Contribute?

### Reporting Bugs

If you come across a bug, please open an issue on GitHub. Please include as much detail as possible, including:
- A clear and descriptive title.
- A step-by-step description of how to reproduce the bug.
- The browser and operating system you are using.
- Any relevant screenshots or error messages.

### Suggesting Enhancements

If you have an idea for an enhancement, feel free to open an issue to discuss it. 

### Code Contributions

At this time, I am not actively seeking code contributions. This is a small, personal project. However, if you have a great idea, feel free to fork the repository and experiment.

## Styleguides

The project uses a simple structure with vanilla JavaScript, HTML, and CSS. Please maintain the existing coding style and conventions.
